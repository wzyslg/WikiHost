# WikiHost微基主机怎么样？WikiHost介绍和测评

## WikiHost 介绍
WikiHost 是一家老牌主机商，由原来的 50OVZ 和 50KVM 合并而来，主要销售高速 VPS 和虚拟主机，数据中心位于中国香港、韩国、日本和美国等地，服务面向全球用户，特别适合个人用户和中小型企业。
